## Week Six: Electronic Output Devices <p> This week we learned about electronic output devices, the opposite of what we did last week. Electronic output devices take information and output data in the form of sensors, like the LED strips or the music box sensor. The assignment for this week was to use an output sensor that you have never used before. After learning about all of my different options, I decided yo go with an LED strip because I wanted to use it within my final project. </p> <p> This week was actually perfect for planning my final project because the I was going to have to use both the input and output sensors. Since I was already planning on using them later in the semester, this was just really preparation to get the code and everything working. My idea was to have my tepmperature input scanner (from last week) change the color out LED output strips based on the temperature. This relates back to my final project because I want to have the lights on my display be based off of a user's bosy temperature. </p> <p> In order to start this whole process, I first had to solder wires to my LED strip, and hotglue the solder down: </p> ![image of solder](./solder.jpeg) ![image of solder](./solder2.jpeg) ![image of solder](./solder3.jpeg) ![image of solder](./solder4.jpeg) <p> This was the code that I used to collect my data (all credit goes to Nathan's website!!!!!): </p> <pre><code class="language-arduino"> int analog_pin = A3; int tx_pin = 4; int read_high; int read_low; int diff; void setup() { pinMode(tx_pin, OUTPUT); // Pin 4 provides the voltage step Serial.begin(9600); } void loop() { digitalWrite(tx_pin,HIGH); // Step the voltage high on conductor 1. read_high = analogRead(analog_pin); // Measure response of conductor 2. delayMicroseconds(100); // Delay to reach steady state. digitalWrite(tx_pin,LOW); // Step the voltage to zero on conductor 1. read_low = analogRead(analog_pin); // Measure response of conductor 2. diff = read_high - read_low; // desired answer is the difference between high and low. After looking at the sample codes on the website, I decided to use a thermistor to detect changes in temperature. The circuit was not too hard to build because there was diagrams shown on the website (credit to Nathan's website!) </p> ![image of making circuits](./makingcircuits3.jpeg) ![image of making circuits](./makingcircuits4.jpeg) <p> This was the code I used for this sensor (once again all the credit for programming goes to Nathan!!!!): </p> <p> The only thing I changed with this code was I added "ln" after the print statement to see the lines for my graph. </p> <pre><code class="language-arduino"> int ThermistorPin = A0; int Vo; float R1 = 10000; float R2, T; float A = 3.354e-03; float B = 2.5698e-4; void setup() { Serial.begin(9600); } void loop() { Vo = analogRead(ThermistorPin); R2 = R1 * 1/(1023.0 / (float)Vo - 1.0); //Calculate resistance of thermistor from voltage divider math. T = T - 273.15; //Convert from Kelvin to Celcius. // Serial.print("Temperature: "); Serial.println(T); // Serial.println("C"); delay(500); } </code></pre> <p> I used the same process (pretty much) with this sensor in finding the minimum and maximum values.
