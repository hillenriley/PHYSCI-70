## Temperatus Lux #### Completed Final Project Documentation <p> My final project is the Temperatus Lux. In Latin, this means "tmperature light" and it is just that. This light is able to measure the surrounding air temperature through a thermistor and output that data through an LED light display.</p> ### Completed Project Video <p> https://www.youtube.com/watch?v=ofMs3E71IWQ </p> ### Materials Included <p> - Arduino UNO Board</p> <p> - Bread Board</p> <p> - Male to Male wires</p> <p> - Female to Male wires</p> <p> - Thermistor (preferribly with long wires)</p> <p> - Plywood</p> <p> - Acrylic Sheet (preferribly clear)</p> <p> </p> ### Arduino LED Code <pre><code class="language-arduino"> define PIN 6 define NUMPIXELS 60 int ThermistorPin = A0; int Vo; float R1 = 10000; float R2, T; float A = 3.354e-03; float B = 2.5698e-4; Adafruit_NeoPixel strip(NUMPIXELS, PIN, NEO_GRB + NEO_KHZ800); void setup() { Serial.begin(9600); strip.begin(); // INITIALIZE NeoPixel strip object (REQUIRED) strip.show(); // Turn OFF all pixels ASAP strip.setBrightness(100); // Set BRIGHTNESS low to reduce draw (max = 255) } void loop() { Vo = analogRead(ThermistorPin); R2 = R1 * 1 / (1023.0 / (float)Vo - 1.0); //Calculate resistance of thermistor from voltage divider math.
